08 / NOTES

Faça & não faça

Diferenças deliberadas em relação ao DS editorial: estética operacional troca expressão por consistência e eficiência.
✓ Faça
  • — Use Inter Tight para tudo que é ação ou label.
  • — Use mono em qualquer número, ticker, código ou data.
  • — Use o azul de sistema como única cor de ação.
  • — Use cores semânticas só para dado financeiro e estado.
  • — Mantenha tabelas sem zebra; use hover para localização.
  • — Prefira ícones em outline simples (1px, 14–16px).
✕ Não faça
  • — Não use Cormorant em interface de produto.
  • — Não use ouro para botões, links ou estados.
  • — Não use sombras pronunciadas — eleva com 1px line.
  • — Não use raios maiores que 8px.
  • — Não misture verde e ouro na mesma tela.
  • — Não use itálico serifado; reserve para a comunicação institucional.
